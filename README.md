# coopel

Instructions

Install Node.js



Download repository
`git clone git@github.com:thalescm/coopel.git`

Install Node 
http://nodejs.org/download/

Instal NPM
https://docs.npmjs.com/getting-started/installing-node

Install MongoDB
http://docs.mongodb.org/manual/installation/

Open Terminal

run `mongod`  

In another tab,

run
`npm install -g bower`

run
`bower install`

run
`npm install`

run
`npm install -g grunt cli`

run
`grunt serve`
